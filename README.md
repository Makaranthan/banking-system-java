# Simple Banking System (Java)

## Description
A CLI-based Banking System implemented in Java using Object-Oriented Programming. 
Users can create an account, deposit, withdraw, and check their account balance.

## Features
- Create a new bank account
- Deposit money into account
- Withdraw money from account
- Check current account balance

## Technologies Used
- Java
- Object-Oriented Programming (Classes, Methods, Objects)
- Scanner for user input

## How to Run
1. Compile the Java file:javac SimpleBank.java
   2. Run the program:java SimpleBank


## Learning Outcome
- Practiced OOP fundamentals like methods and object creation
- Built a simple real-world application using Java
- Strengthened logical thinking and problem-solving skills
